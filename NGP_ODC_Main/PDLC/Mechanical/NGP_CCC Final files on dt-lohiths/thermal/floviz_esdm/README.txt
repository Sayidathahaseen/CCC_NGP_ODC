This CD/DVD contains the files required to install this release. You can 
find installation instructions in the FloVIZ Installation Instructions 
document, available on this CD/DVD in the "release_documents" directory.

This CD/DVD also contains release highlights and additional documentation to 
help you with the installation and licensing of Mentor Graphics tools. The 
following documents are available at the top level of this CD/DVD or in the 
"release_documents" directory and can be accessed before you install your 
software:

+ FloVIZ Release Highlights
  =========================
  Filename: FloVIZ_9_2_rh.pdf
  Contains detailed information regarding application delivery and support, 
  licensing, installation, configuration requirements, and an overview of 
  the product updates.
  
+ FloVIZ Installation Instructions
  ================================
  Filename: release_documents/FloVIZ_9_2_install.pdf
  Contains quick installation information.

 
You can view these PDF files using Acrobat Reader 4.0.5 or newer. If
needed, you can download Acrobat Reader from www.adobe.com.
